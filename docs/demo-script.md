# 同尘｜2分30秒 Demo Script

## 0:00–0:20｜问题

**旁白**

生活已经很难了，但打开社交网络，我们还是要面对争论、比较和表演。与此同时，很多人感到孤单，只是想分享一顿饭、一片天空，却不知道该发给谁。

**画面**

同尘封面和一句话定位：“分享一点生活，被合适的人轻轻接住。”

## 0:20–0:38｜产品答案

**旁白**

同尘是一个 AI 驱动的无压力分享空间。它不把每条内容广播给整个世界，而是先理解你，再投递生活碎片，并让合适的人轻轻接住。

**画面**

指向侧边栏“演示模式”，展示中文与英文切换。

## 0:38–1:08｜GPT-5.6 理解

**操作**

点击“放下一点”，输入一段分享，再点击“让 AI 理解这段分享”。

**旁白**

用户不用填写人格问卷，只需要自然地表达。这里通过 OpenAI Responses API 调用 GPT-5.6 Terra，把分享解析成意图、情绪与回应边界。结果在发布前可见、可确认；AI 帮助理解人，但不替人定义自己。

## 1:08–1:40｜投递与接住

**操作**

进入“微光”，打开一条晚餐分享，指出“为什么来到你这里”和“希望被怎样接住”。点击“我看见了”，再打开“放下一点”。

**旁白**

每份生活碎片只投递给少量边界相合的人。发布者可以选择只想被看见、欢迎一句回应，或者不需要回复。接收者不会先评价，而是看见、共鸣，或分享自己的平行故事。

## 1:40–1:58｜求同存异

**操作**

打开“遇见”。指向“你们的相同”和“你们的不同”。

**旁白**

同尘不会把相同的人锁在一起。共同点提供安全感，适量差异带来新的视角。我们不显示匹配度，也不催促用户建立关系。

## 1:58–2:17｜安全

**操作**

打开“此刻的我”，暂停一项理解，再点击安全保护演示。

**旁白**

个性化越深入，保护就越重要。内部画像永不公开；系统限制陌生账号快速接近，并阻断索取位置、联系方式或钱财等越界行为。

## 2:17–2:30｜Codex 与结尾

**旁白**

Codex 帮助我们把产品洞察收束成完整闭环，并完成双语界面、交互、安全演示、GPT-5.6 接入和持续部署。我们的目标不是增加停留时长，而是让更多分享，以用户期待的方式被接住。在同尘，分享一点生活，不必面对整个世界。

**画面**

同尘品牌画面、Demo URL、核心闭环“理解 · 投递 · 接住”。

## Recording notes

- 使用 1440×900 或 1920×1080 录制。
- 浏览器缩放保持 100%，关闭个人书签和通知。
- 鼠标移动放慢，操作后停顿约半秒。
- 中文版本为主，切换英文约 3 秒证明国际化能力。
- 录制前必须在托管环境配置 OpenAI 密钥，确认画面显示 GPT-5.6 Terra，而不是 Demo fallback。
- 不连续滚动；每个页面只展示与旁白对应的一个重点。
